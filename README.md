# Medical RAG System (Simple Version)

This project is a lightweight Retrieval-Augmented Generation (RAG) system for answering basic medical questions using semantic search and sentence embeddings.

## 🔍 Project Idea
The system retrieves relevant medical information based on user queries and generates simplified explanations.

## ⚙️ How it works
1. Input question
2. Convert text into embeddings
3. Compute similarity with medical knowledge base
4. Retrieve top relevant documents
5. Generate a simple answer

## 🧠 Technologies Used
- Python
- SentenceTransformers
- NumPy

## 📌 Example
Query: high blood pressure  
Answer: Hypertension is high blood pressure and can damage the heart.

## 🚀 Goal
This project demonstrates basic NLP, semantic search, and RAG pipeline concepts for research and PhD applications.
